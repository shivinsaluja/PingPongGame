# PingPongGame
This is a 2 player Ping Pong Game made using two languages namely Javascript And HTML. 

## Instructions

Player 1 Paddle - Up -"w"
                - Down -"s"
Player 2 paddle - Up -"ArrowUp"
                 -Down -"ArrowDown"
                                  
If the ball misses your paddle the other player wins. Game over.
